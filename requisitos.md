# Documento de Requisitos - PetMatch v1.0

## 1. Visão Geral
O PetMatch é um sistema web fictício criado para conectar abrigos de animais e possíveis adotantes, facilitando o processo de adoção de pets.

## 2. Requisitos Funcionais
* **RF01:** O sistema deve permitir o cadastro de usuários (Adotantes e Abrigos).
* **RF02:** O sistema deve permitir que abrigos cadastrem pets disponíveis para adoção, incluindo fotos, idade e temperamento.
* **RF03:** O sistema deve permitir que adotantes busquem pets por filtros (espécie, porte, cidade).
* **RF04:** O sistema deve permitir que o adotante clique em "Quero Adotar", enviando uma notificação ao abrigo.

## 3. Requisitos Não Funcionais
* **RNF01:** O sistema deve ser responsivo (acessível via celular e desktop).
* **RNF02:** Senhas de usuários devem ser armazenadas com criptografia.
